# tslox

TSLox is an interpreter for the lox programming language, implemented in TypeScript.

## Usage

### REPL Mode

To start the interactive REPL:

```bash
tslox
```

### Executing lox Files

To run a lox source file:

```bash
tslox path/to/your/script.lox
```
